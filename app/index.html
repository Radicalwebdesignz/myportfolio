My Workflow

I divide each project into four follow-up areas: comprehensive research, wireframing, design and development. This process allows me to create great product with client goals in mind.

Approach

My approach to website design is to create immersive and beautiful web applications through user-centric design. The audience should have an enjoyable, positive experience when using the website.

UX/UI Design

I sketch page layouts, wire-frames, sitemaps and menu structures transforming ideas into well thought out design concepts ensuring ease of use and a personalised look and feel that are a joy to use.

Code

Once the design has been finalised, the development process starts by taking the graphical elements defined in the design process and writing clean and efficient code using web industry standards.

What I Do

HTML5 - CSS3 - Bootstrap - JavaScript - jQuery - Ajax - PHP - MySQL - Responsive Design - WordPress - SEO

On the front-end, I use a combination of HTML, CSS, and JavaScript to build everything a user sees and interacts on a website. On the back-end I develop the application, server, and database using PHP MySQL that make up the foundational structure of a website centered around LAMP stack.

Apart from these hard-core technical skills, I enjoy using Adobe Photoshop and Adobe XD. I use build tools such as Git, Gulp, NPM and variuos Gulp Plugins to streamline the development process.
